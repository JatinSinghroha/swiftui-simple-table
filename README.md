## üìù Description

SwiftUI table view, which works on **iOS** and **macOS**.

- ‚úÖ Simple to use (check out included examples).
- ‚úÖ Displays **all columns** on iOS, even when the horizontal size class is compact.
- ‚úÖ Supports vertical and **horizontal** scrolling (or no scrolling at all if that's what you need).
- ‚úÖ Supports **"sticky" headers** (with `.simpleTableHeader()` view modifier).
- ‚úÖ Allows configuring column/row size: equal heights, equal widths, or constrained aspect ratio.
